QGIS Plugin to load SaveEcoBot data into QGIS

MIt diesem Plugin werden die Radioaktivitätsdaten von Saveecobot (https://www.saveecobot.com/en/radiation-maps) heruntergeladen. Der Download benötigt mehrere MInuten, da die Detaildaten, die auch die Stationsinformationen enthalten, einzeln heruntergeladen werden müssen. Dazu sind etwa 700 (!) Anfragen erforderlich. Die verwendeten URLs sind bereits passend voreingestellt, können im Falle einer Änderung der Webseite aber angepasst werden.
